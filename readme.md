# 3d Renderer

My custom render pipeline (cpu based, for now).

Includes:
- sdl3
- c++17
- ???
- profit

## install deps

1. `git clone https://github.com/libsdl-org/SDL.git --branch release-3.4.0 vendored/SDL`
2. Change architecture of your OS in `cmakelists.txt`
3. `cmake -S . -B build`

## Run

1. make
2. ./build/renderer
